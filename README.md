# System Requirements
 * ubuntu 18.04 LTS
 * drawlib
 * Xlunch
 
# Build・Compile・Run
```
$ git clone https://github.com/kanekoNK/RanGame.git
$ cd RanGame
$ gcc -lm ISGM.c -ldrawlib
$ ./a.out
```
